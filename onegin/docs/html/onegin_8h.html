Подробное описание

Данный файл содержит в себе прототипы функций для чтения/записи романа в стихах "Евгений Онегин" и трагедии "Гамлет".

Заметки
Если файл с произведением для сортировки содержит кириллические символы, то он обязательно должен быть в кодироке iso88595. В противном случае, не гарантируется надлежащее сравнение символов файла. Если же файл содержит только символы латиницы, он может быть в любой 8-битной кодировке.
