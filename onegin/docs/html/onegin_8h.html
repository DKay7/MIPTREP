Функция paritation разделяет массив вокруг опорного элемента.

Данная функция используется внутри функции qsort, как часть алгоритма быстрой сортировки.

Аргументы
[in,out]arrayУказатель на массив указателей для сортировки.
[in]n_membКоличество элементов в массиве.
[in]el_sizeРазмер одного элемента в байтах.
[in,out]leftУказатель на левую границу сортируемого участка (индекс элемента массива).
[in,out]rightУказатель на правую границу сортируемого участка (индекс элемента массива).
[in]pivot_indexИндекс опорного элемента.
[in]comparatorУказатель на функцию для сравнения элементов массива.
Возвращает
Возвращает индекс последнего элемента левой части массива.
