◆ unit_test()

void unit_test ( const char *  path)

Функция для проведения юнит-тестов

Файл с тестами должен иметь следующуюю структуру:

  1. В первой строке файла содержится число n – количество тестов. Далее идут n строк с тестами непосредственно.

  2. Каждая строка имеет вид: коэффициент_а коэффициент_b коэффициент_c количество_корней верный_x1 верный_x2

    Коэффициенты a, b, c – числа с плавающей точкой – коэффициенты в уравнении вида ax^2 + bx + c

    Количество корней – целое число – соответствующий код из solver_return_codes

    Верные ответы x1, x2 – числа с плавающей точкой, в случае если один или оба корня отсутствуют либо уравнение имеет бесконечно много корней, нужно передать NAN в качестве значения верного ответа, соответственно, либо только для x1 или x2, либо сразу для обоих

Аргументы
[in]pathПуть к файлу с тестами
