Подробное описание

Заголовочный файл с прототипами функций для решения квадратного уравнения

Данный файл содержит в себе прототипы функций, используемых для решения квадратного уравнения и вывода результата

◆ solver_return_codes

enum solver_return_codes

Перечисление кодов, которые функции solve_square_equasion и solve_linear_equasion возвращают в зависимости от количества решений

Элементы перечислений
INFINITY_VALUE_ERROR 

Ошибка, при которой одна из переменных приняла значение INFINITY.

SOLVER_DEFAULT 

Значение по умолчанию

INFINITY_NUMBER_OF_SOLUTIONS 

Бескончно много решений (x - любое)

NO_REAL_SOLUTIONS 

Нет действительных решений

ONE_SOLUTION 

Одно действительное решение

TWO_SOLUTIONS 

Два действительных решения
